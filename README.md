# Asn1PKCS

PKCS #1/PKCS #8 RSA key ASN.1 encoder/decoder for C#.

## Requirements

- .NET Standard 1.3

## Installing via NuGet

```
PM> Install-Package Nyamikan.Asn1PKCS
```
